完善博客实用功能

发表于 2018-05-12 | 分类于 实用技术 | |
字数统计: 912 | 阅读时长 ≈ 4

既然自己搭建了一个博客系统，就要有点样子的，进行个性化的设置，好在hexo next主题实现这些功能简直不要太容易，于是就跟风来一波，主要是是通过配置两个文件实现的：
站点配置文件..\blog\_config.yml
主题配置文件..\blog\themes\next\_config.yml

设置标签和分类

添加标签

第一次配置，输入命令：

hexo new page tags

确认站点配置文件里有tag_dir: tags
确认主题配置文件里有tags: /tags
编辑站点的source/tags/index.md，添加以下代码

title: tags
date: 2015-10-20 06:49:50
type: "tags"
comments: false

添加分类

第一次配置，输入命令：

hexo new page categories

确认站点配置文件里有category_dir: categories
确认主题配置文件里有categories: /categories
编辑站点的source/categories/index.md，添加以下代码

title: categories
date: 2015-10-20 06:49:50
type: "categories"
comments: false

每当创建新的md文件时候，输入

hexo new newfile.md

然后在生成文件的开头添加
